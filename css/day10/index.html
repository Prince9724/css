display : flex

The display: flex property in CSS initiates the Flexible Box Layout Module, commonly known as Flexbox. When applied to an HTML element, it transforms that element into a flex container, and its direct children become flex items.

Display : Flex, Flex - Warp

The CSS flex-wrap property, used within a flexbox layout, controls whether flex items are forced onto a single line or allowed to wrap onto multiple lines when there isn't enough space in the flex container.
