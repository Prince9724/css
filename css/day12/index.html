display: flex(child property) flex-basis

  • The flex-basis CSS property specifies the initial size of a flexible item before any remaining space is distributed. It's essentially the main size of the item.

flex-basis Auto
flex-basis 20px
flex-basis:34%

display: flex(child property) flex-Grow

  • A flex-grow value of 1 is the default for most browsers and means the item will take up an equal share of the extra space.

  • flex-grow is a CSS property for flex children that specifies how much a flexible item will grow relative to the other flexible items inside the same flex container when there is available space. It's a unitless number that serves as a proportion.

flex-basis Auto
flex-grow : 1;
flex-basis:2;

display: flex(child property) flex-shrink

  • flex-shrink is a CSS property for flex children that defines how a flexible item will shrink relative to other items inside the same flex container when there isn't enough space. It's a unitless number that serves as a proportion.

flex-shrink :0;
flex-shrink :0;
flex-shrink :2;

display: flex(child property) aling-self : flex-start;

  • The align-self CSS property allows you to override the alignment set by align-items for an individual flex item. While align-items sets the default alignment for all items along the cross-axis, align-self provides fine-grained control for a single item.

  • When you apply align-self: flex-start to a flex item, it aligns that specific item to the start of the cross-axis. This overrides whatever align-items is set to on the parent flex container.
